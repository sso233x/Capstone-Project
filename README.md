### To Start Project

### Navigating to directories
**You will need two terminal windows open, 
one for the frontend and one for the backend.**

1. **Start the Back-End first**
   cd backend
   
   **To Start use**
   flask run/app.py

3. **Start the Front-End**
   cd frontend

   **To Start use**
   npm start

### The app should start in your browser
